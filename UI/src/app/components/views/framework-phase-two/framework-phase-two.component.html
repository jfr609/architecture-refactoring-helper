Phase 2

In "Phase 2: Strategy Definition", a migration strategy and refactoring approach for the identification of microservices is selected.

As a first step, you may want to explore the list of all cataloged approaches and tools by clicking on "Explore all approaches" or "Explore all tools". It will present a list with the possibility to inspect all details of an approach or tool. Each approach represents a peer-reviewed scientific publication.

"Find recommended approaches" offers basic filter presets for Refactor, Rebuild, and New Application, and allows for a detailed filtering by subsequently clicking on "Configure". "Find scenario-based recommended approaches" preselects the detailed filter configuration with the qualities specified for the scenarios in "Phase 1: System Comprehension", and allows to apply further filters.

Explore all approaches
Freely explore all refactoring approaches that exist in the repository.
Explore all tools
Freely explore all tools that exist in the repository.
Find recommended approaches
Find refactoring approach recommendations based on qualities, preferred inputs, outputs, processes, and usability aspects.
Find scenario-based recommended approaches
Find refactoring approach recommendations based on the scenarios specified in "Phase 1: System Comprehension".
